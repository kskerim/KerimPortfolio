# Portfolio

Développement du portfolio React + TypeScript. Fournis les maquettes / captures et on va coller exactement au design (typographies, grille, couleurs, animations).

## Démarrage

1. Installer les dépendances:
```
npm install
```
2. Lancer le serveur de dev:
```
npm run dev
```

## Prochaines étapes une fois la maquette reçue
- Définir palette (CSS variables)
- Config typographie (font-face ou Google Fonts)
- Système de spacing & grille responsive
- Composants atomiques (Button, Heading, Text, SectionContainer)
- Layout navigation + sections (Hero, À propos, Compétences, Projets, Parcours, Contact)
- Animations (framer-motion ou CSS) selon la maquette
- SEO de base / meta / OpenGraph
- Optimisation Lighthouse

Envoie les images quand prêtes.
